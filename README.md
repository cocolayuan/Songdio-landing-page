# Songdio Landing Page

Songdio 官网落地页项目，按版本分目录管理。

## 目录结构

```
Songdio-landing-page/
├── V1-testing/   # V1 测试版（Hero 视频 + 音乐轮播）
├── V2/           # V2 开发中（当前活跃版本）
└── README.md
```

## 快速开始

进入对应版本目录后安装依赖并启动：

```bash
cd V2
npm install
npm run dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000)。

## 版本说明

| 版本 | 状态 | 说明 |
|------|------|------|
| V1-testing | 归档 | 首版原型：全屏 Hero 视频 + 音乐卡片轮播 |
| V2 | 开发中 | 新版落地页 |
